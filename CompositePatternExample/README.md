# Composite Pattern

## Working

Composes objects into tree structures and treats single objects and groups uniformly.

## Advantages

- Simplifies tree handling
- Uniform client API
- Easy to add new components

## Disadvantages

- Can blur object responsibilities
- Harder to restrict certain children

## Usage

- File systems
- UI hierarchies
- Menu trees

`App.java` is the module entry point.
