# Adapter Pattern

## Working

Wraps an incompatible interface and converts it into the shape the client expects.

## Advantages

- Reuses existing code
- Keeps client code unchanged
- Useful for legacy integration

## Disadvantages

- Adds an extra wrapper
- Can hide design mismatch instead of fixing it

## Usage

- Legacy API integration
- Third-party libraries
- Interface compatibility layers

## Example

`App.java` is the module entry point.
