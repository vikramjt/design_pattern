# Visitor Pattern

## Working

Adds new operations to a structure without changing the classes being visited.

## Advantages

- Easy to add operations
- Keeps related logic outside the data classes
- Good for object graphs

## Disadvantages

- Hard to add new element types
- Can break encapsulation if overused

## Usage

- AST processing
- Reporting over object structures
- Document/object graph traversal
